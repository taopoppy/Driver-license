# 交通标志的分类和作用

+ 交通标志按照功能分为**主标志**和**辅助标志**，后者一般设置在前者下面，对其辅助说明
+ 主标志分为：**警告标志**、**禁令标志**、**指示标志**、**指路标志**、**旅游区标志**、**作业区标志**、**告示标志**

## 警告标志
+ 警告标志的作用：**警告车辆，行人前方有危险，谨慎通过** ✔

+ **黄底，黑边，黑图形，为顶点朝上的等边三角形或者矩形**

+ 交叉路口标志
  + **十字交叉路口标志**（十字形状）：前方有十字路口，请谨慎慢行 ✔
  + **环形交叉路口标志**（三个环绕的箭头）：前方有环形交叉路口，谨慎慢行 ✔
  + **T形交叉路口标志**（一个大写字母T）：前方有T形交叉路口，谨慎慢行 ✔✔✔
  + **Y形交叉路口标志**（类似于大写字母Y的路口分叉图形）：前方有Y形交叉路口，谨慎慢行✔

+ 路况标志
  + **左右急转弯**（左右60度的弯曲箭头）：前方有有左右急转弯（急弯标志会和速度标志一起使用，标志最高车速）✔✔✔✔✔
  + **反向弯路标志**（倾斜的Z字母）：前方有两个连续方向相反的急转弯 ✔
  + **连续弯路标志**（笔直波浪线）：前方有连续三个或者三个以上方向相反的弯路 ✔
    （口诀:一急二反三连续：急转弯路，反向弯路，连续弯路）
  + **陡坡标志**（黑色上坡黄色箭头朝上的图像、黑色下坡黄色箭头朝下）：前方有向上的陡坡路段或者向下的陡坡路段 ✔✔✔✔✔
  + **连续下坡标志**（连续两个下坡的箭头）：前方有两个及以上的下坡路段 ✔✔
  + **事故易发路段标志**（小汽车追尾大卡车的标志）：事故易发路段（不是易发生追尾路段） ✔✔✔
  + **注意危险标志**（一个叹号）：
  + **窄路标志**（下胖上窄）✔✔✔✔
  + **窄桥标志**（上下宽，中间窄）✔✔✔
  + **易滑标志**（车打滑的标志）✔✔
  + **注意行人标志**（人走在人行横道）✔
  + **注意儿童标志**（两个孩子在上学）：经常有儿童出入，并非一定是学校区域 ✔✔
  + **注意非机动车标志**（一辆自行车）✔✔
  + **注意残疾人标志**（一个残疾人坐着轮椅）✔
  + **注意横风标志**（类似于一个断头的路灯）：前方有很强的侧风向 ✔✔
  + **注意牲畜标志**（一头牛）✔
  + **注意野生动物标志**（一头鹿） ✔
  + **隧道标志**（隧道）✔
  + **隧道开车灯标志**（一辆车进入隧道前开灯的图像）✔✔
  + **注意信号灯标志**（红绿灯）✔✔
  + **注意落石标志**（石头从半坡落下来）：提示前方是落石的傍山路段，有左右之分 ✔✔
  + **傍山险路标志**（山坡上有路的图像）：傍山险路，谨慎驾驶 ✔✔
  + **村庄标志**（屋子旁边有棵松树）✔✔
  + **堤坝路段**（一辆车马上掉入水中）：有左右之分 ✔
  + **渡口标志**（一辆车在一条船上）：前方是车辆渡口，谨慎驾驶 ✔✔
  + **过水路面，漫水标志**（一辆车溅起水花） ✔✔
  + **慢行标志**（慢字）✔
  + **驼峰桥标志**（一个拱形桥）：前方是拱度很大，影响视距的拱形桥 ✔✔✔
  + **路面不平标志**（波浪路面）：路面颠婆 ✔✔
  + **路面高突标志**（一个凸起的路面）✔
  + **路面低洼标志**（一个凹进去的路面）✔
  + **施工标志**（工人正在铲东西）✔
  + **有人看守铁路道口标志**（一个栅栏）✔
  + **无人看守铁路道口标志**（一个火车标志）✔✔
  + **叉形符号标志**（箭头上面一个叉号）：多股铁路和道路相交✔
  + **注意障碍物标志**（绕开障碍的箭头）✔✔✔✔✔✔
  + **注意合流标志**（不同的路合成一个箭头）：前方有车辆汇入进来 ✔
  + **双向交通标志**（反向相反的两个箭头）：前方不分离双向行驶路段，注意会车 ✔✔✔
  + **注意潮汐车道标志**（双向箭头两边有虚线）：前方车辆行驶方向随交通管理变化 ✔
  + **注意保持车距标志**（车尾有两个双箭头）：保持车距，前方经常发生追尾 ✔✔
  + **注意分离式道路标志**（两个左右箭头）：前方是分离式道路 ✔
  + **前方车辆排队标志**（三个小汽车）
  + **注意路面结冰标志**（一个雪花）
  + **注意雨雪天标志**（乌云下雨）
  + **注意雾天标志**（一个雾字）
  + **注意不利气象标志**（一个乌云打闪电）
  + **避险车道标志**（上面有避险车道四个字）


+ 辅助标志
  + **斜杠符号标志**（一个红色斜杠在无人看守铁路道口标志的下方）：表示距道口的距离，一条红色横杠为50米 ✔✔
  + **建议速度标志**（具体的速度）（注意建议速度是黄底黑字，限速是红圈白底黑字）✔

## 禁令标志
+ 禁令标志的作用：**禁止，限制和相应解除，道路驾驶者应该严格遵守**
+ 禁令标志的图像：**红圈，红杠，黑图像，圆形或者八边形或者倒三角形**

+ 禁止标志
  + **停车让行标志**（一个停字）：车辆必须在停车线前停车瞭望，确认安全后才能通过
  + **减速让行标志**（一个让字）：慢行或者停车，观察干道情况，让干道车辆先行
  + **会车让行标志**（方向相反的红黑箭头）：会车先行或者会车让行取决于上箭头是黑还是红
  + **禁止通行标志**（白圈）：前方禁止一切车辆和行人通行 ✔✔
  + **禁止驶入标志**（一条白色的横杠）：前方道路禁止一切车辆驶入 ✔✔

  + **禁止停车标志**（红叉蓝底）：禁止车辆临时或者长时间停放✔✔✔
  + **区域禁止停车标志**（红叉蓝底，下有区域二字）：禁止车辆在区域内临时或者长时间停放
  + **解除区域禁止停车标志**（区域禁止停车标志上有五条黑色反向横线）：禁止车辆在区域内临时或者长时间停放

  + **禁止长时停车标志**（左上右下红色斜杠，蓝底）：在限制的范围内，禁止长时间停放车辆，临时停车不受限制✔✔
  + **禁止区域长时停车标志**（左上右下红色斜杠，蓝底，最下面有区域二字）：在限制的范围内，禁止长时间停放车辆，临时停车不受限制
  + **区域长时停车解除标志**（禁止区域长时停车标志上有五条反向黑色横线）

  + **停车检查标志**（黑色横杠下由检查字样）：前方是机动车必须接收检查的地方
  + **禁止机动车驶入标志**（黑色轿车）：禁止各类机动车驶入
  + **禁止载货汽车驶入标志**（黑色货车）：前方路段禁止载货汽车驶入
  + **禁止电动三轮车驶入标志**（黑色三轮车）：前方路段禁止电动三轮车驶入
  + **禁止大型客车驶入标志**（黑色大型客车）：前方路段禁止大型客车驶入
  + **禁止小型客车驶入标志**（黑色小型客车）：前方路段禁止小型客车驶入
  + **禁止挂车或者半挂车驶入标志**（黑色挂车或者半挂车）：前方路段禁止挂车或者半挂车驶入
  + **禁止拖拉机驶入标志**（黑色拖拉机）：前方路段禁止拖拉机驶入
  + **禁止三轮汽车，低速货车驶入标志**（黑色三轮汽车）：前方路段禁止三轮汽车，低速货车驶入
  + **禁止摩托车驶入标志**（黑色人骑着摩托车）：前方路段禁止摩托车驶入
  + **禁止某两种车辆驶入**（具体看禁止哪两种车辆）：前方禁止两种车辆驶入
  + **禁止非机动车驶入标志**（黑色自行车）：前方路段禁止非机动车驶入
  + **禁止畜力车车驶入标志**（黑色马拉车）：前方路段禁止畜力车驶入
  + **禁止人力客运三轮车驶入标志**（黑色人力客运三轮车）：前方路段禁止人力客运三轮车驶入
  + **禁止人力货运三轮车驶入标志**（黑色人力货运三轮车）：前方路段禁止人力货运三轮车驶入
  + **禁止人力车驶入标志**（黑色人力推车）：前方路段禁止人力车驶入
  + **禁止行人进入标志**（黑色人在走路）：前方路段禁止行人进入
  + **禁止左右转弯后者向左向右**（左右箭头）
  + **禁止直行，禁止直行和左转弯，禁止直行有右转弯**（向上箭头和左右箭头结合）
  + **禁止掉头标志**（弯曲的箭头）
  + **禁止鸣喇叭**（黑色喇叭）
  + **禁止超车标志**（黑色超车标志）：前方路段禁止超车，直到前方出现解除禁止超车标志
  + **解除禁止超车标志**（反向五条黑色斜线）：可以超车
  + **禁止运输危险物品车辆驶入标志**（车顶爆炸）：前方路段禁止运输危险品车辆驶入
  + **海关标志**（黑色横杠上有海关两个字）：前方是海关，停车检查

+ 限制标志
  + **限制宽度标志**（具体的数字）：前方路段禁止装载宽度超过数字的车辆通行
  + **限制高度标志**（具体的数字）：前方路段禁止装载高度超过数字的车辆通行
  + **限制质量标志**（具体的数字t）：前方路段禁止装载质量超过数字的车辆通行
  + **限制轴重标志**（车轴上有具体的数字）：前方路段禁止轴重超过数字的车辆通行
  + **限制速度标志**（具体的数字）：前方路段禁止超过限制速度
  + **解除速度限制标志**（具体的数字外加反向五条黑色斜线）：可以超过这个速度了
  + **区域限制速度标志**（具体的数字下面有区域两个字）
  + **解除区域限制速度标志**（具体的数字下面有区域两个字，还有五条黑色反向斜线）

（临时停车是临时上下客，装货卸货，驾驶人在车内或者车旁等候）

## 指示标志
+ 指示标志的作用：**是指示车辆，行人按照标志指示的路线，方向行驶**
+ 指示标志的颜色为：**蓝底白图案，形状为圆形，长方形和正方形**

+ **直行标志**（白色向上箭头）：一切车辆只准直行 ✔✔
+ **左右转弯标志**（白色转弯箭头）：一切车辆只准向左向右转弯 ✔✔
+ **直行和左转弯标志，直行和右转弯标志**（前方车辆只能向前，向左，向右转弯）✔✔
+ **靠左右侧道路行驶**（左右下箭头）：靠左右侧行驶✔✔
+ **立体交叉执行和左右转弯行驶标识**（前方立体交叉口，一切车辆可以直行或者按图线左右转弯行驶）✔✔
+ **环岛行驶标志**（三个旋转的箭头）：前方环形车道靠右行驶，进入环岛，应该让环岛内的车辆先行
+ **单行路标志**（竖直长方形的向上箭头）：前方道路为单向行驶 ✔✔✔✔✔
+ **步行标志**（打人领着小孩）前方道路只供步行，车辆不准进入 ✔✔
+ **鸣喇叭标志**（白色喇叭）车辆行驶到该处应该鸣喇叭提示对向车辆加速慢行
+ **最低限速标志**（白色具体数字）前方路段最低限速（经常在路口和最高时速配合使用） ✔✔
+ **路口优先通行标志**（类似火箭）交叉口主要道路上的车辆享有优先通行权 ✔✔✔✔✔
+ **会车先行标志**（红白反向箭头）会车优先通行 ✔✔
+ **人行横道标志**（黑色小人走黑色斑马线）前方人行横道，减速慢行 ✔✔
+ **左转，右转，直行车道**（方向箭头两边有虚线）✔✔
+ **直行和左右合用车道**（方向为直行和左右合用车道标志）✔✔✔✔✔✔
+ **掉头车道，左转和掉头车道** ✔
+ **分向行驶车道标志**（前面标志的大杂烩）各车道车辆按照指示在规定的车道行驶 ✔✔✔
+ **允许掉头标志**（U型箭头） ✔
+ **多乘员车辆行驶车道**（小车上坐着两个人）：车道只供多乘员的车辆行驶 ✔✔
+ **公交路线专用车道**（公交图像）：公交车专用车道
+ **快速公交系统专用车道**（BRT公交车道）

+ **机动车行驶标志**（小汽车标志）：前方道路只供机动车行驶
+ **机动车车道标志**（小汽车车道标志）：指示车道只供机动车行驶

+ **非机动车行驶标志**（自行车加右下箭头）：前方道路只供非机动车行驶
+ **非机动车车道标志**（自行车车道标志）：指示车道只供非机动车行驶

+ **停车位标志**（蓝底，白色P字母）：机动车允许停放区域，需要和停车位线一同使用 ✔✔✔✔
  + **停车位有很多变种，根据辅助图像来辨识具体意思**

## 指路标志
+ 指路标志的作用是：**指引道路信息，为驾驶者带来位置距离信息**
+ 一般城市的指路标志为：**蓝底白图形，白边框**，高速公路上的为：**绿底。白图形，白边框**

+ 一般道路标志（**路径指引标志**和**地点指引标志**和**道路沿线设施指引标志**和**其他路径信息指引标志**）
  + **路径指引标志**包括：
    + **交叉路口预告标志**
    + **交叉路口告知标志**
      + **十字路口交叉标志**
      + **丁字路口交叉标志**
      + **Y型交叉路口标志**
      + **环形交叉路口标志**
      + **互通式立体交叉路口标志**
      + **分叉标志**
      + **确认标志**（确认当前的道路信息和方向信息）

  + **地点指引标志**
    + **地名标志**
    + **著名地点标志**
    + **分界标志**
    + **地点识别标志**（重要场所的识别和指向）

  **道路沿线设施指引标志**
    + **停车场标志**
    + **错车道标志**（会车需要让车，这里有避让的场所，辅助标志表示前方错车道的距离）
    + **人行天桥标志和人行地下道标志**
    + **残疾人设施位置**
    + **观景台标志**（有观景台三个大字）
    + **应急避难场所标志**
    + **休息区标志**（有休息区三个大字）
    + **绕行标志**（绕行路线为黑色）
    + **此路不通标志**（前方有红色障碍）
    + **车道数变少变多标志**
    + **交通监控设备标志**（一个蓝色摄像头）
    + **隧道出口距离预告标志**
    + **线性诱导标志**（这个是红底白线，向上为两侧通信，右下为右侧通信，左下为左侧通信）
    + **里程碑，里程牌，百米桩**（国G道用红字，省S道用蓝字，县X道乡Y道使用黑字）
    + **公路界碑**

  **其他路径信息指引标志**
    + **起点终点标志**（高速公路的起点和终点）
    + **终点提示标志**（有一个红色的斜线和起点做区分）
    + **停车领卡标志**
    + **车距确认标志**
    + **特殊天气建议速度标志**（公路前面有一层雾）
    + **紧急电话和救援电话标志**（紧急电话只有电话这个图像，救援图像有救援二字）
    + **收费站预告和收费站标志**
    + **ETC车道指示标志**（电子不停车收费车道）
    + **计重收费标志**
    + **加油站标志**
    + **紧急停车带标志**
    + **服务区预告标志**（服务区有刀叉标志）
    + **停车区预告标志**（停车区没有刀叉标志）
    + **爬坡车道标志**
    + **超载超限监测站标志**
    + **方向标志**（东南西北）

## 其他标志
  + 旅游区标志的作用和含义：吸引人们到旅游区玩
  + 旅游标志分为：**指引标志**和**旅游符号标志**，一般为棕色
  + **指引标志**分为：
    + **指引距离标志**
    + **指引方向标志**
  + **旅游符号标志**一般不是景区，而是一些运动或者娱乐活动的图形，划水，高尔夫等标志

  + **作业区标志的作用**：通知道路交通阻断，绕行
    + **作业区的标志在各种类型的图案中都有，结合其他图案一起理解**
    + **施工标志**
    + **改道标志**（结合改道和前方道路不通的标志的结合）
    + **绕行标志**

  + **告示标志的作用**： 解释，指引道路设置，路外设施，就这有关法律的东西，一般为白底黑字
    + **地点告示**
    + **眼睛酒后驾车标志**
    + **严禁乱认器物标志**
    + **驾驶时禁用手机标识**
    + **急弯减速（下坡）标识**
    + **大型车靠右标识**
    + **系安全带标识**
    + **校车停靠站点标识**

  + **辅助标识**
    + **时间，方向，距离等等**
    + **教练路线**
